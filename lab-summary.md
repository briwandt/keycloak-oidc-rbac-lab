# Keycloak OIDC RBAC Lab Summary

## Objective

Demonstrate authentication and authorization using Keycloak, OpenID Connect (OIDC), JWT tokens, and Role-Based Access Control (RBAC).

## Components

- Keycloak Identity Provider
- OIDC Client (demo-app)
- Realm: identity-lab
- Users:
  - alice (user)
  - bob (manager)

## Validation

Successfully:

- Created realm
- Configured OIDC client
- Created users
- Assigned roles
- Requested access tokens
- Decoded JWT payloads
- Verified role claims in tokens
