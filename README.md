# Keycloak OIDC RBAC Lab

This project demonstrates:

- Keycloak authentication
- OpenID Connect (OIDC)
- JWT token validation
- Role-Based Access Control (RBAC)

## Users

- alice (manager)
- bob (analyst)

## Setup

1. Start Keycloak
2. Create realm: identity-lab
3. Create users and roles
4. Obtain access tokens
5. Validate role claims in JWTs
